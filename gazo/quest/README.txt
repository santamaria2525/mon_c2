Quest フォルダ - クエスト機能関連画像
======================

このフォルダには、クエスト機能に関連する画像が含まれています。

主要な画像:
- quest*.png - クエスト画面関連
- normal_quest.png - 通常クエスト
- obu*.png - オーブ関連
- fire*.png - 炎関連
- shugo_*.png - 守護関連
- solo.png, multi.png - ソロ・マルチ選択
- minnato.png - 皆と（マルチ参加）
- sanka.png - 参加ボタン
- shohi20.png - 消費20
- kyube*.png - キューブ関連
- kyukyoku*.png - 究極関連
- tiket*.png - チケット関連
- tenshi_*.png - 天使関連
- seed.png - シード関連
- zokuse.png - 属性関連
- battle.png - バトル関連
- start.png - スタート関連

これらの画像は、クエスト機能の実行時に使用されます。
