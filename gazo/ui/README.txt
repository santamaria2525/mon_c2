UI フォルダ - 汎用UI要素
===================

このフォルダには、アプリケーション全体で共通して使用されるUI要素の画像が含まれています。

主要な画像:
- ok.png, close.png, yes.png, no.png - 基本的なボタン
- back.png, retry.png, modoru.png - ナビゲーションボタン
- settei.png, option.png - 設定関連ボタン
- search.png, kensaku.png - 検索関連ボタン
- name*.png - 名前関連UI
- friends*.png - フレンド関連UI
- doors*.png - ドア関連UI
- gand*.png - ガンダム関連UI

これらの画像は複数の機能で共通して使用されるため、機能横断的な配置となっています。
