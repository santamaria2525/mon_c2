Mission フォルダ - ミッション機能関連画像

このフォルダには、ミッション機能に関連する画像が含まれています。

主要な画像:
- m_mission*.png - ミッション画面関連
- m_close.png - ミッション閉じる
- m_*tai.png - *体関連（3体、4体、5体等）
- m_mitatsu.png - 未達成
- m_stjoho.png - ST情報
- m_tokugacha.png - 特別ガチャ
- m_tujo*.png - 通常関連
- nm_*.png - 新ミッション関連
- uketoru.png - 受け取り
- uketsuke.png - 受付
- m_kakunin.png - 確認
- m_tassei.png - 達成
- m_uke*.png - 受け取り関連

これらの画像は、ミッション機能の実行時に使用されます。
